# Пайплайн сбора данных

## Принцип

**Sports API** — источник правды для фактов. **Perplexity** — только контекст. **Сервер** — расчёты. **Claude** — только анализ.

---

## Шаг 1 — Определение матча (Sports API)

Пользователь обязательно выбирает спорт (⚽🏒🏀) → убирает неоднозначность, определяет эндпоинт.

- **Конкретный матч** ("ЦСКА — Спартак"): `teams?search=` × 2 → `fixtures/headtohead?h2h={id1}-{id2}&next=1`
- **Одна команда** ("ЦСКА"): `teams?search=` → `fixtures?team={id}&next=1` → подтверждение

## Шаг 2 — Структурированные данные (Sports API, параллельно)

| Запрос | Эндпоинт | Данные |
|---|---|---|
| Последние 10 матчей × 2 | `fixtures?team={id}&last=10` | Результаты, счета, дома/гости |
| H2H | `fixtures/headtohead?h2h={id1}-{id2}&last=5` | 5 личных встреч |
| Травмы | `injuries?fixture={id}` | Травмы, дисквалификации |
| Коэффициенты | `odds?fixture={id}` | Коэфы букмекеров |
| Таблица | `standings?league={id}&season={year}` | Позиция, очки |
| Статистика сезона × 2 | `teams/statistics?team={id}&league={id}&season={year}` | Владение, удары, голы |

~10 запросов параллельно, 2-3 сек.

## Шаг 3 — Контекст (Perplexity, параллельно с шагом 2)

Один запрос:
```
Матч: {home} vs {away}, {competition}, {date}.
Ответь кратко:
1. За что борется каждая команда? Приоритет матча?
2. Ротация состава? Важный матч до/после?
3. Трансферы/тренерские изменения за 2 недели?
4. Судья? Средние ЖК и пенальти за сезон?
5. Погода на {date} в {city} в {time}?
```

## Шаг 4 — Расчёты (сервер, мгновенно)

| Метрика | Формула |
|---|---|
| Средние голы | sum / N за последние 5-10 |
| Форма (W/D/L) | Последние 5 результатов |
| Серия | Одинаковые результаты с конца |
| Форма дома/выезд | Фильтр по venue |
| BTTS % | Матчи где обе забили / всего |
| Тотал Б2.5 % | Матчи с 3+ голов / всего |
| Clean sheets | Матчи с 0 пропущенных |

**xG**: на MVP — Perplexity ("средний xG за 5 матчей по Understat/FBref"). После MVP — Sportmonks xG add-on.

## Шаг 5 — Анализ (Claude API Sonnet)

Claude получает JSON → 7 секций отчёта (см. [`../ui/report-sections.md`](../ui/report-sections.md)). Стрим через SSE (см. [`sse.md`](sse.md)).

## API-Sports: стоимость

- **Бесплатно**: 100 запросов/день (~10 отчётов/день). Все эндпоинты, навсегда.
- **$10/мес**: 7500 запросов/день (~750 отчётов/день).

## Таймлайн

| Время | Что происходит |
|---|---|
| 0-2 сек | Определение матча |
| 2-5 сек | Sports API + Perplexity параллельно |
| 5-6 сек | Расчёты, начало стриминга |
| 6-15 сек | Claude анализ, секции появляются |
| ~15 сек | Отчёт готов |

## Fallback

| Ситуация | Действие |
|---|---|
| Команда не найдена | Альтернативные написания → сообщаем |
| Нет матча | "Нет запланированных матчей" |
| Sports API недоступен | Fallback на Perplexity (пометка о неточности) |
| Perplexity не отвечает | Отчёт без контекста |
| xG недоступен | Пропускаем xG-блок |
| Нет коэффициентов | Скрываем секцию коэфов |

# VSKRYTIE / ВСКРЫТИЕ

Telegram Mini App — аналитический помощник для беттора.

Пользователь выбирает спорт и лигу, называет команду — получает структурированный отчёт с данными и рекомендацию (исход + тотал). Сервис не говорит «ставь» — даёт данные, оценку уверенности и обоснование. Решение за пользователем.

## Ключевой принцип

Букмекерские коэффициенты — самый точный предиктор исхода (подтверждено исследованиями). Мы не строим альтернативную модель. Мы делаем три вещи:

1. **Собираем линии** от нескольких букмекеров (API-Sports) → находим консенсус и расхождения
2. **Добавляем контекст** (травмы, вратарь, ротация) через Perplexity → ищем то, что букмекеры могли учесть с задержкой
3. **Добавляем форму** (последние 5 матчей из API-Sports) → объясняем пользователю почему коэфы такие

Claude анализирует всё вместе → исход, тотал, confidence.

## Как строится рекомендация

1. **Коэффициенты = основа** — implied probability определяет фаворита
2. **Контекст = корректировка** — свежие факторы (травмы, вратарь) могут сдвинуть оценку
3. **Форма = объяснение** — показывает *почему* линии такие

Мы не пытаемся быть умнее рынка. Claude ищет ситуации, где свежий контекст ещё не полностью отражён в линиях.

## Отчёт включает (MVP)

- Коэффициенты от букмекеров + implied probability + расхождения
- Контекст (травмы, вратарь, ротация, медиа-контекст)
- Форма (последние 5 матчей каждой команды)
- Рекомендация с confidence (high/medium/low)

## Стек

| Слой | Технология |
|---|---|
| App | Next.js + Tailwind (Telegram Mini App) |
| БД | Supabase |
| Спортивные данные | API-Sports (api-sports.io) — результаты, коэффициенты |
| Контекст | Perplexity API (травмы, вратарь, ротация) |
| AI анализ | Claude API (Sonnet) |
| Оплата (MVP) | Telegram Stars |

## Принципы

- **API-Sports для чисел** — точные данные без галлюцинаций
- **Perplexity только для контекста** — то, что API не покрывает
- **Коэфы = основа предикшна** — не пытаемся быть умнее рынка
- **Конфиги по лигам** — новая лига = новый конфиг, не новый код
- **КХЛ — стартовая точка**, архитектура готова к любым лигам
- **Два рынка ставок** на старте (исход + тотал)
- **Минимум данных для MVP** — только то, что влияет на рекомендацию

# VSKRYTIE / ВСКРЫТИЕ

Telegram Mini App — аналитический помощник для беттора.

Пользователь выбирает спорт и лигу, называет команду или матч — получает структурированный отчёт с фактами и рекомендацию (2 ставки с обоснованием). Сервис не говорит "ставь" или "не ставь" — даёт данные и оценку, решение за пользователем.

## Отчёт включает

- Турнирное положение и мотивация команд
- Форма (последние 5 матчей)
- История противостояний (H2H) в сезоне
- Сезонная статистика (зависит от вида спорта)
- Кадры, травмы, медиа-контекст
- Коэффициенты от 5 букмекеров
- Итоговая рекомендация с опорой на факты

## Стек

| Слой | Технология |
|---|---|
| App | Next.js + Tailwind (Telegram Mini App) |
| БД | Supabase |
| Спортивные данные + контекст | Perplexity API (все данные) |
| AI анализ | Claude API (Sonnet) |
| Оплата (MVP) | Telegram Stars |

## Принципы

- **Никаких Sports API** — всё через Perplexity
- **Конфиги по лигам** — новая лига = новый конфиг, не новый код
- **КХЛ — стартовая точка**, архитектура готова к любым лигам и видам спорта
- **Только два рынка ставок** на старте (исход + тотал)

# VSKRYTIE / ВСКРЫТИЕ

Telegram Mini App — аналитический помощник для беттора.

Пользователь называет команду или матч — получает отчёт и короткую рекомендацию (2–4 предложения) на основе данных. Сервис не говорит "ставь" или "не ставь" — даёт факты и оценку, решение за пользователем.

## Отчёт включает

- Форма команды, состав, травмы
- Серия побед / поражений
- История противостояний (H2H)
- Коэффициенты российских букмекеров (Фонбет, Winline, PARI, BetBoom)
- Итоговая рекомендация с опорой на факты

## Стек


| Слой                                | Технология                             |
| ----------------------------------- | -------------------------------------- |
| App                                 | Next.js + Tailwind (Telegram Mini App) |
| БД                                  | Supabase                               |
| Спортивные данные                   | API-Sports (api-sports.io)             |
| Контекст (мотивация, погода, судья) | Perplexity API                         |
| AI анализ                           | Claude API (Sonnet)                    |
| Оплата (MVP)                        | Telegram Stars                         |

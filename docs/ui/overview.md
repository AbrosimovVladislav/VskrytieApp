# UI — Обзор экранов

## Карта

Home (поиск) → Analyzing (стриминг) → Report (отчёт)
+ History (список отчётов) + Balance (покупка) + Settings

Навигация: bottom tab bar — Главная, Анализы, Баланс, Ещё.

---

## Home

- Табы спорта: ⚽ Футбол, 🏒 Хоккей, 🏀 Баскетбол (по умолчанию — футбол)
- Поле ввода ("ЦСКА — Спартак", "Реал завтра")
- CTA "Анализировать"
- Примеры запросов под спорт (тап = автозаполнение)
- 2-3 последних запроса
- Индикатор остатка отчётов (⚡ 5), тап → Balance

## Analyzing

- Прогресс-бар + пошаговые статусы
- Skeleton-заглушки, секции появляются по мере стриминга
- SSE события: см. [`../arch/sse.md`](../arch/sse.md)

## Report

- Вертикальный скролл, 7 секций-карточек
- Появление: fade-in + slide-up, 200ms
- Кнопка Share в шапке
- **Детали секций**: [`report-sections.md`](report-sections.md)

## History

- Карточки прошлых отчётов: команды, лига, дата, рекомендация + confidence dots
- Тап → полный отчёт

## Balance

- Остаток отчётов крупно
- Пакеты за Telegram Stars (5/10/30)
- Поток оплаты: [`../arch/payments.md`](../arch/payments.md)

---

## Микровзаимодействия

**Haptics**: light на CTA, medium на готовый отчёт, error на ошибку, success на оплату.

**Анимации**: секции fade-in + slide-up 200ms, skeleton → контент fade 150ms, typing cursor 500ms, confidence dots заливка 100ms.

**Дизайн-токены**: [`../design.md`](../design.md)

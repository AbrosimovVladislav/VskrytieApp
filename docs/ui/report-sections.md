# 7 секций отчёта

Каждая секция = карточка `bg-bg-card rounded border shadow`. Дизайн-токены: [`../design.md`](../design.md).

---

## 1. Контекст матча

Лига, тур/стадия, дата, время, стадион, гербы команд.
Плашка мотивации: high=зелёный, medium=жёлтый, low=красный + причина текстом.

## 2. Форма и результаты

**Форма**: dots последних 5 — ● победа (зелёный), ◐ ничья (серый), ○ поражение (красный). Рекорд (3W 1D 1L) + серия. Подсекция дома/выезд.

**H2H**: крупные цифры побед (7 : 3), пропорциональный бар, 3 последних встречи (дата + счёт).

Текст анализа от Claude (2-3 предложения).

## 3. Продвинутая статистика

**xG-перформанс** (ключевой блок):
- голы > xG → "▲ Забивают больше xG" (warning, регрессия)
- голы < xG → "▼ Забивают меньше xG" (positive, могут прибавить)
- голы ≈ xG → "≈ В рамках ожиданий"

**Сравнительные бары**: голы забитые/пропущенные, xG, удары в створ, владение %, BTTS %, тотал Б2.5 %. Лидер — accent, отстающий — bg-inner. Для "меньше = лучше" — инверсия.

Текст инсайта от Claude (2-3 предложения).

## 4. Кадровая ситуация

"✓ Все в строю" (text-positive) или список потерь:
- 🔴 key / 🟡 rotation / ⚪ minor
- Имя, позиция, причина, срок возвращения

Текст от Claude (1-2 предложения).

## 5. Контекстные факторы

Плашки в сетке 2 колонки: погода, дни отдыха, судья (имя + ЖК + пенальти), трансферы. Нет данных — плашка скрыта.

Текст от Claude (1-2 предложения).

## 6. Коэффициенты

Таблица: 2-3 букмекера × П1/X/П2. Строка среднего. Лучший коэф — text-accent.

Value-бейджи: 🟢 "Коэф занижен", ⚪ "В рынке", 🔴 "Коэф завышен".

Текст от Claude (1-2 предложения).

## 7. Рекомендация

Выделенный блок (bg-bg-card-dark). Summary стримится с typing-cursor.

**Confidence**: dots ●○ + текст (Высокая/Средняя/Низкая).

**1-3 карточки рынков**: название, value-бейдж, reasoning.

**Рынки**: 1X2, тотал Б/М, BTTS, фора, угловые Б/М, карточки Б/М.

Дисклеймер: "Решение всегда за вами. ВСКРЫТИЕ предоставляет аналитику, не финансовые рекомендации."

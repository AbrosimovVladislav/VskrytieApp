## Screen 1 — Home / Search

**Цель**: быстро принять запрос, создать ощущение "аналитика на связи".

```
┌────────────────────────────────┐
│  ВСКРЫТИЕ            [⚡ 5]    │  ← font-display лаймовый + остаток отчётов
├────────────────────────────────┤
│                                │
│   Введите команду или матч     │  ← subtitle (text-muted)
│                                │
│  ┌──────────────────────────┐  │
│  │ 🔍  ЦСКА — Спартак       │  │  ← input: bg-bg-card border-border rounded-[10px]
│  └──────────────────────────┘  │
│                                │
│  ┌──────────────────────────┐  │
│  │     АНАЛИЗИРОВАТЬ        │  │  ← CTA: bg-accent text-bg-card-dark rounded-[10px]
│  └──────────────────────────┘  │
│                                │
│  Примеры запросов:             │  ← подсказки (text-muted)
│  · Реал Мадрид завтра          │
│  · Манчестер Сити — Арсенал    │
│  · ЦСКА хоккей                 │
│                                │
│  ── Недавние ──────────────    │  ← 3 последних запроса
│  ЦСКА — Локомотив    вчера     │
│  Барселона — ПСЖ    3 дня     │
│                                │
└────────────────────────────────┘
│  🏠 Главная │ 📊 Анализы │ 💰 Баланс │ ⚙️ Ещё  │  ← bottom nav
```

**Детали:**
- Заголовок: `font-display text-2xl text-accent`
- Input: `bg-bg-card border border-border rounded-[--radius-button] px-4 py-3.5`
- CTA кнопка: `bg-accent text-bg-card-dark font-semibold rounded-[--radius-button] h-12`
- `⚡ 5` — иконка + число отчётов, тап → переход на Balance
- Подсказки: text-muted, тап → автозаполнение

## Screen 2 — Analyzing (Загрузка / Стриминг)

**Цель**: показать прогресс, снять тревогу ожидания 15–20 сек.

```
┌────────────────────────────────┐
│  ←  ЦСКА — Спартак             │  ← back + запрос
├────────────────────────────────┤
│                                │
│  ━━━━━━━━━━━━░░░░░░░  62%     │  ← прогресс-бар bg-accent
│                                │
│  ✓  Определяем матч            │  ← done (text-positive)
│  ✓  Собираем статистику        │  ← done
│  ●  Анализируем данные...      │  ← active (animate-pulse accent)
│  ○  Формируем отчёт            │  ← pending (text-muted)
│                                │
│  ┌──────────────────────────┐  │
│  │  ░░░░░░░░░░░░░░░░░░░░░░  │  │  ← skeleton bg-bg-card animate-pulse
│  │  ░░░░░░░░░░░░░░░░░░░░░░  │  │
│  └──────────────────────────┘  │
│                                │
└────────────────────────────────┘
```

**Стриминг-поведение (прогрессивное раскрытие):**

| Этап | Событие SSE | Что показываем |
|---|---|---|
| 1 | `step:identify` | "Определяем матч" ✓ |
| 2 | `step:stats` | "Собираем статистику" ✓ |
| 3 | `step:analyze` | "Анализируем данные" ● |
| 4 | `section:header` | Header секция (fade-in) |
| 5 | `section:form` | Form секция |
| 6 | `section:stats` | Stats секция |
| 7 | `section:injuries` | Injuries секция |
| 8 | `section:h2h` | H2H секция |
| 9 | `section:odds` | Odds секция |
| 10 | `section:recommendation` | AI-рекомендация + haptic |

**Детали:**
- Прогресс-бар: `bg-accent`, анимированный fill
- Steps done: `text-positive`, active: `bg-accent animate-pulse`, pending: `bg-border-secondary`
- Skeleton: `bg-bg-card animate-pulse rounded-[--radius-card]`

## Микровзаимодействия и Haptics

| Момент | Haptic | Описание |
|---|---|---|
| Нажатие CTA | `impactOccurred('light')` | Мягкий отклик |
| Секция появляется | `impactOccurred('light')` | Каждая секция |
| Отчёт готов | `impactOccurred('medium')` | Финальный аккорд |
| Ошибка | `notificationOccurred('error')` | Вибро-ошибка |
| Успешная оплата | `notificationOccurred('success')` | Подтверждение |

---

## Анимации

| Элемент | Анимация |
|---|---|
| Секция появляется | `fade-in + slide-up`, 200ms ease-out |
| Skeleton → контент | `fade-in`, 150ms |
| Typing в рекомендации | `cursor blink`, 500ms interval |
| Прогресс-бар | `width transition`, 300ms linear |
| Карточка tap | `scale 0.98 → 1.0` |
| Табы switch | `bg transition`, 200ms |

# Секция: Мотивация

Турнирное положение и мотивация каждой команды.

## Данные

Из `AnalysisReport.motivation`:

```ts
{
  data: {
    team1: string;  // "5-е место, борьба за плей-офф, высокий приоритет"
    team2: string;  // "2-е место, борьба за лидерство конференции"
  },
  analysis: string; // "ЦСКА мотивирован больше — каждое очко критично..."
}
```

## Макет

Две плашки (одна под другой), каждая — карточка с данными команды. Под ними — блок анализа.

```
┌──────────────────────────────┐
│ ЦСКА                         │
│ 5-е место · Западная конф.   │
│ Борьба за плей-офф           │
└──────────────────────────────┘
┌──────────────────────────────┐
│ СКА                          │
│ 2-е место · Западная конф.   │
│ Борьба за лидерство          │
└──────────────────────────────┘

💬 "ЦСКА мотивирован больше..."
```

- Название команды: `font-semibold text-[16px]`
- Детали: `text-text-secondary text-[14px]`
- Анализ: `text-text-secondary text-[14px] italic` в AI-блоке

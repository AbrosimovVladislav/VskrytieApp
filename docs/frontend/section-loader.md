# Секция: Лоадер pipeline

Отображает прогресс выполнения pipeline пока пользователь ждёт.

## Визуал

Вертикальный stepper — точки, соединённые линиями:

```
● Поиск матча              ← пройден (accent)
│
● Коэффициенты             ← пройден (accent)
│
◉ Контекст                 ← текущий (пульсация)
│
○ Форма команд             ← ожидает (muted)
│
○ Анализ AI
```

## Состояния точки

| Состояние | Стиль |
|---|---|
| Пройден | `bg-accent` — заполненный круг |
| Текущий | `bg-accent animate-pulse` — с пульсацией |
| Ожидает | `border border-muted` — пустой круг |

## Линия между точками

- Пройденная: `bg-accent`
- Ожидающая: `bg-border`

## Данные от оркестратора

Компонент получает `currentStep: number` (1–5) и рендерит состояние.

## Поведение

- При переходе на новый шаг — плавная анимация (заполнение точки + линии)
- Когда все 5 шагов пройдены — переход к отображению отчёта

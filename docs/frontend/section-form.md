# Секция: Форма команд

Последние 5 матчей каждой команды.

## Данные

Из `AnalysisReport.form`:

```ts
{
  data: {
    team1_last5: GameResult[];
    team2_last5: GameResult[];
  },
  analysis: string;
}
```

## Макет

Для каждой команды — ряд из 5 точек (Win/Loss/OT) + список матчей.

```
ЦСКА: ● ● ○ ● ●  (3В 1П 1ОТП)

20.03  Динамо (Д)     3:2  W
18.03  Локомотив (В)  1:3  L
15.03  Спартак (Д)    4:2  W
...

СКА:  ● ● ● ○ ●  (4В 1П)
...
```

## Точки результатов (dots)

| Результат | Цвет |
|---|---|
| W (победа) | `bg-positive` |
| L (поражение) | `bg-negative` |
| OTW (победа в ОТ) | `bg-positive opacity-60` |
| OTL (поражение в ОТ) | `bg-negative opacity-60` |

- Точки: круги 8px в ряд
- Матчи: список, `text-[14px]`, дом/выезд в скобках
- (Д) = дома, (В) = выезд
- Анализ Claude внизу секции

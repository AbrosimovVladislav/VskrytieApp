# Секция: H2H (история встреч)

Все матчи между командами в текущем сезоне.

## Данные

Из `AnalysisReport.h2h`:

```ts
{
  data: {
    games: H2HGame[];  // { date, score, venue }
  },
  analysis: string;
}
```

## Макет

Список встреч в сезоне:

```
Встречи в сезоне (3)

15.11  ЦСКА 2:1 СКА     Арена ЦСКА
03.12  СКА 3:3 ЦСКА      Ледовый дворец
20.01  ЦСКА 1:4 СКА     Арена ЦСКА
```

- Счёт: выделен `font-semibold`
- Арена: `text-text-secondary text-[12px]`
- Анализ Claude внизу секции

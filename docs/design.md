# VSKRYTIE — Design System

## Aesthetic Direction

**Dark / Analytical / Sport**

Вдохновение: ESPN, Sofascore, спортивные дашборды.
Цель: максимум информации при минимуме декора. Данные — главный герой.

---

## Color Tokens

```css
@theme {
  /* Фон */
  --color-bg: #0a0a0f;            /* основной фон */
  --color-bg-secondary: #13131a;  /* фон панелей, bottom nav */
  --color-bg-card: #1a1a24;       /* фон карточек */

  /* Акцент — зелёный (позитивные метрики, форма, успех) */
  --color-accent: #00c853;
  --color-accent-dim: rgba(0, 200, 83, 0.15);

  /* Предупреждение — оранжевый (риски, травмы, негатив) */
  --color-warning: #ff6d00;
  --color-warning-dim: rgba(255, 109, 0, 0.15);

  /* Текст */
  --color-text: #ffffff;
  --color-text-secondary: rgba(255, 255, 255, 0.6);
  --color-muted: #6b6b80;

  /* Границы */
  --color-border: rgba(255, 255, 255, 0.08);
  --color-border-accent: rgba(0, 200, 83, 0.3);

  /* Шрифт */
  --font-sans: "Poppins", sans-serif;
}
```

## Typography

| Роль | Вес | Размер | Использование |
|---|---|---|---|
| Заголовок страницы | 600 (SemiBold) | `text-xl` | Название экрана |
| Заголовок карточки | 600 (SemiBold) | `text-base` | Название матча/команды |
| Тело текста | 400 (Regular) | `text-sm` | Описания, параграфы |
| Числа/статистика | 600 (SemiBold) | `text-2xl` | Счёт, коэффициенты |
| Метка / badge | 500 (Medium) | `text-xs` | Теги статусов |

Числа всегда с `font-variant-numeric: tabular-nums` для выравнивания.

## Компоненты

### Карточка отчёта (ReportCard)

```
┌──────────────────────────────────┐
│ ЦСКА vs Спартак          23 мар  │
│ ─────────────────────────────────│
│ Форма: ✓ ✓ ✗ ✓ ✓       [ЗЕЛЁНЫЙ]│
│ Итог: Преимущество хозяев        │
└──────────────────────────────────┘
```

- Фон: `bg-card` (`#1a1a24`)
- Радиус: `rounded-xl`
- Padding: `p-4`
- Граница: `border border-border`
- Hover: `border-border-accent` + `bg-accent-dim`

### Секция отчёта (ReportSection)

```
┌──────────────────────────────────┐
│ 📊 ФОРМА КОМАНДЫ                 │
│ ─────────────────────────────────│
│  ЦСКА:    П П П П П   (5 побед)  │
│  Спартак: П П П Н П   (4 победы) │
└──────────────────────────────────┘
```

- Заголовок секции: `text-xs text-muted uppercase tracking-wide`
- Разделитель: `border-b border-border`

### StatsBadge

```
┌─────────┐  ┌─────────┐  ┌──────────┐
│  5П 0П  │  │  2.10   │  │ -3 игрока│
│  Форма  │  │ Коэф.П1 │  │  Травмы  │
└─────────┘  └─────────┘  └──────────┘
```

- Зелёный badge: `bg-accent-dim text-accent border-border-accent`
- Оранжевый badge: `bg-warning-dim text-warning`
- Нейтральный badge: `bg-bg-secondary text-text-secondary`

### Блок рекомендации (Recommendation)

```
┌──────────────────────────────────┐
│ ⚡ ИТОГОВАЯ ОЦЕНКА               │
│ ─────────────────────────────────│
│ ЦСКА выглядит фаворитом: 5 побед │
│ подряд дома, Спартак без 2 ключе-│
│ вых игроков. Коэффициент 1.85 на │
│ победу хозяев выглядит ценным.   │
└──────────────────────────────────┘
```

- Фон: `bg-accent-dim`
- Граница: `border border-border-accent`
- Текст: `text-text` (белый, не зелёный)

## Bottom Navigation

4 вкладки:
1. **Главная** — поле ввода
2. **История** — прошлые отчёты
3. **Баланс** — оставшиеся отчёты / покупка Stars
4. **Настройки** — язык и проч.

Стиль: `bg-bg-secondary border-t border-border h-16`
Активная вкладка: `text-accent`
Неактивная: `text-muted`

## Состояния загрузки

### Skeleton отчёта

Пока Claude стримит — показывать заполненные секции одну за другой:
1. Заголовок с командами (сразу)
2. Skeleton строки для каждой секции (пока данные идут)
3. Секции появляются по мере получения стрима

Цвет skeleton: `bg-bg-card animate-pulse`

### Индикатор анализа

```
Анализируем ЦСКА vs Спартак...
● Собираем статистику
● Анализируем данные
● Формируем отчёт
```

Прогресс через SSE events от API route.

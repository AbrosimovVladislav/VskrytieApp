# Промпт: Поиск матча (шаг 1)

**API**: Perplexity

## Шаблон

```
Какой ближайший матч {query} в {league.name} сезона {league.season}?
Дата, время (московское), соперник, место проведения.
```

## Переменные

| Переменная | Источник |
|---|---|
| `{query}` | Запрос пользователя |
| `{league.name}` | `LeagueConfig.name` |
| `{league.season}` | `LeagueConfig.season` |

## Пример (КХЛ)

```
Какой ближайший матч ЦСКА в КХЛ сезона 2025/2026?
Дата, время (московское), соперник, место проведения.
```

## Ожидаемый ответ

Конкретный матч: две команды, дата, время, арена. Если не удалось — варианты для уточнения.

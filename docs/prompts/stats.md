# Промпт: Статистика сезона (шаг 5)

**API**: Perplexity

## Шаблон

Формируется динамически из `leagueConfig.statsFields`.

```
Статистика {team1} и {team2} в {league.name} сезона {league.season}: {statsFieldsList}.
```

Где `{statsFieldsList}` — human-readable перечисление из `statsFields`.

## Маппинг statsFields → текст промпта

| Ключ | Текст в промпте |
|---|---|
| position | позиция в таблице |
| points | очки |
| goalsFor | голы забитые за сезон |
| goalsAgainst | голы пропущенные за сезон |
| avgGoalsFor | средние голы за матч |
| avgGoalsAgainst | средние пропущенные за матч |
| powerPlayPct | реализация большинства (%) |
| penaltyKillPct | игра в меньшинстве (%) |
| penaltyMinutes | средние штрафные минуты за матч |
| shotsPerGame | средние броски за матч |

## Пример (КХЛ)

```
Статистика ЦСКА и СКА в КХЛ сезона 2025/2026: позиция в таблице, очки, голы забитые/пропущенные за сезон, средние голы за матч, реализация большинства (%), игра в меньшинстве (%), средние штрафные минуты за матч, средние броски за матч.
```

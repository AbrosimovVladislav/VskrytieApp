# Промпт: Букмекерские линии (шаг 7)

**API**: Perplexity

## Шаблон

Список букмекеров берётся из `leagueConfig.bookmakers`.

```
Коэффициенты на матч {team1} vs {team2}, {league.name}, {date}.
Линии от: {bookmakersList}.
Для каждого: П1 в основное время, ничья, П2 в основное время, тотал больше 4.5, тотал меньше 4.5.
```

## Переменные

| Переменная | Источник |
|---|---|
| `{team1}`, `{team2}` | `MatchInfo` |
| `{league.name}` | `LeagueConfig.name` |
| `{date}` | `MatchInfo.date` |
| `{bookmakersList}` | `LeagueConfig.bookmakers.join(", ")` |

## Пример (КХЛ)

```
Коэффициенты на матч ЦСКА vs СКА, КХЛ, 25 марта 2026.
Линии от: Фонбет, Винлайн, PARI, Олимпбет, Лига Ставок.
Для каждого: П1 в основное время, ничья, П2 в основное время, тотал больше 4.5, тотал меньше 4.5.
```

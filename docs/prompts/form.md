# Промпт: Форма команд (шаг 3)

**API**: Perplexity

## Шаблон

```
Последние 5 матчей {team1} и последние 5 матчей {team2} в {league.name} сезона {league.season}.
Для каждого: дата, соперник, счёт, дома или на выезде.
```

## Переменные

| Переменная | Источник |
|---|---|
| `{team1}`, `{team2}` | `MatchInfo` |
| `{league.name}` | `LeagueConfig.name` |
| `{league.season}` | `LeagueConfig.season` |

## Пример (КХЛ)

```
Последние 5 матчей ЦСКА и последние 5 матчей СКА в КХЛ сезона 2025/2026.
Для каждого: дата, соперник, счёт, дома или на выезде.
```

# Промпт: Контекст и мотивация (шаг 2)

**API**: Perplexity

## Шаблон

```
Матч {team1} vs {team2}, {league.name}, {date}.
Какое турнирное положение каждой команды?
За что борется каждая (плей-офф, место в таблице, вылет)?
Какая стадия сезона?
Какой приоритет этого матча для каждой?
```

## Переменные

| Переменная | Источник |
|---|---|
| `{team1}`, `{team2}` | `MatchInfo` |
| `{league.name}` | `LeagueConfig.name` |
| `{date}` | `MatchInfo.date` |

## Пример (КХЛ)

```
Матч ЦСКА vs СКА, КХЛ, 25 марта 2026.
Какое турнирное положение каждой команды?
За что борется каждая (плей-офф, место в таблице, вылет)?
Какая стадия сезона?
Какой приоритет этого матча для каждой?
```
